## MULTI-SENSE NLP MODEL

### SETUP
1. Clone the repository
   ```bash
   git clone https://github.com/Sachin-0001/MultiSense-NLP.git
    cd MultiSense-NLP
    ```
2. Install the required packages
   ```bash
   pip install -r requirements.txt
   ```
3. Models and artifaces are located in the 'models' folder. Use them with appropriate encodings and transformations as per the notebook
